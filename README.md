# **Helmet-Safety-Protection Project**

## **Mục lục**

I. Giới thiệu
II. Cách cài đặt
III. Sử dụng phần mềm

### **I. Giới thiệu**

- Helmet Safety Protection Project là dự án được xây dựng nhằm phát hiện và xác định người đeo mũ bảo hiểm trong khu vực công nghiệp. 

- Hệ thống sử dụng ML để phân loại và cảnh báo người không đội mũ bảo hiểm

### **II. Cách cài đặt**

1. **Clone repository**

- '''sh
  git clone https://github.com/Landaoduy/Helmet-Safety-Protection.git
  cd Helmet-Safety-Protection
  '''

2. **Tạo môi trường conda**

- '''sh
  conda create -n helmet_safety_env python=3.9
  conda activate helmet_safety_env
  '''

3. **Cài đặt các thư viện cần thiết**

- '''sh
  !pip install -r requirements.txt
  !pip install -e .
  '''
